私について

    • 自己紹介

    情報系の大学でプログラミングやWeb開発を学ぶ中で、Web制作に興味を持ちました。現在はHTMLとCSSの習得に注力し、ポートフォリオ制作を通じてレスポンシブデザインやテンプレートカスタマイズといった実践的なスキルを磨いています。接客業で身につけた円滑なコミュニケーション能力と責任感を武器に、今後はクラウドソーシングでの案件獲得を目指し、実務経験を積んでいきたいと考えています。

    • HTML5 / CSS3
    • Git / GitHub
    • モバイルファーストを意識したレスポンシブ対応
    • デザイン模写・テンプレートカスタマイズ

    • Web制作の実績を積み、フリーランスとしての活動基盤を築く
